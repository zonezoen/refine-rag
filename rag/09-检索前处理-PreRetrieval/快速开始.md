## 3. 快速测试

### 测试 1：查询重写（最简单）

```python
from openai import OpenAI
from os import getenv

client = OpenAI(
    base_url="https://api.deepseek.com",
    api_key=getenv("DEEPSEEK_API_KEY")
)

def rewrite_query(question: str) -> str:
    prompt = f"""请重写以下查询，移除口语化表达：
    
原始查询：{question}
重写后："""
    
    response = client.chat.completions.create(
        model="deepseek-chat",
        messages=[{"role": "user", "content": prompt}],
        temperature=0
    )
    return response.choices[0].message.content.strip()

# 测试
query = "那个，我想知道，嗯，Python怎么用啊？"
print(f"原始：{query}")
print(f"重写：{rewrite_query(query)}")
```

### 测试 2：查询路由

```python
from typing import Literal
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.pydantic_v1 import BaseModel, Field
from langchain_deepseek import ChatDeepSeek

class RouteQuery(BaseModel):
    datasource: Literal["python_docs", "js_docs", "golang_docs"] = Field(
        description="选择最适合的数据源"
    )

llm = ChatDeepSeek(model="deepseek-chat", temperature=0)
structured_llm = llm.with_structured_output(RouteQuery)

system = """你是路由专家。根据问题选择数据源：
- python_docs: Python 相关
- js_docs: JavaScript 相关
- golang_docs: Go 语言相关"""

prompt = ChatPromptTemplate.from_messages([
    ("system", system),
    ("human", "{question}"),
])

router = prompt | structured_llm

# 测试
question = "Python 中的列表推导式怎么用？"
result = router.invoke({"question": question})
print(f"问题：{question}")
print(f"路由到：{result.datasource}")
```

## 5. 常见问题

### Q1: 提示 "No module named 'langchain'"

**解决：**
```bash
pip install langchain langchain-core
```

### Q2: 提示 "DEEPSEEK_API_KEY not found"

**解决：**
1. 确保创建了 `.env` 文件
2. 在代码中加载环境变量：
```python
from dotenv import load_dotenv
load_dotenv()
```

### Q3: 查询分解/HyDE 提示找不到文档

**解决：**
确保有测试数据文件：
- `90-文档-Data/黑悟空/设定.txt`
- `90-文档-Data/黑悟空/黑悟空wiki.txt`

或者修改代码中的文件路径。

### Q4: 语义路由提示 "No module named 'openai'"

**解决：**
```bash
pip install langchain-openai
# 并设置 OPENAI_API_KEY
```

或者改用开源模型：
```bash
pip install langchain-huggingface sentence-transformers
```

### Q5: 运行速度慢

**原因：**
- 首次运行需要下载嵌入模型（如 BGE）
- 模型大小约 100-500MB

**解决：**
等待下载完成，后续运行会快很多。
