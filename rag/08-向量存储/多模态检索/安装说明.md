# Visual-BGE 多模态检索安装说明

## 项目简介

Visual-BGE (Visualized-BGE) 是一个通用的多模态嵌入模型，由 BAAI（北京智源人工智能研究院）开发。它将图像 token 嵌入整合到 BGE 文本嵌入框架中，能够处理超越纯文本的多模态数据。

## 安装步骤

### 1. 克隆 FlagEmbedding 仓库

```bash
# 克隆仓库
git clone https://github.com/FlagOpen/FlagEmbedding.git

# 进入 visual_bge 目录
cd FlagEmbedding/research/visual_bge

# 安装 visual_bge 包
pip install -e .
```

### 2. 安装核心依赖包

```bash
pip install torchvision timm einops ftfy
```

注意：不需要安装 xformer 和 apex，它们对推理不是必需的，而且经常会导致问题。

### 3. 下载模型权重

需要从 Hugging Face 下载预训练模型权重：

**英文模型：**
- 模型名称：`BAAI/bge-visualized-base-en-v1.5`
- 维度：768
- 基础模型：`BAAI/bge-base-en-v1.5`
- 下载链接：https://huggingface.co/BAAI/bge-visualized-base-en-v1.5

**多语言模型（包括中文）：**
- 模型名称：`BAAI/bge-visualized-m3`
- 维度：1024
- 基础模型：`BAAI/bge-m3`
- 下载链接：https://huggingface.co/BAAI/bge-visualized-m3

### 4. 使用 Hugging Face CLI 下载（推荐）

```bash
# 安装 huggingface-hub
pip install huggingface-hub

# 下载英文模型
huggingface-cli download BAAI/bge-visualized-base-en-v1.5 --local-dir ./models/bge-visualized-base-en-v1.5

# 下载多语言模型（中文）
huggingface-cli download BAAI/bge-visualized-m3 --local-dir ./models/bge-visualized-m3
```

## 使用示例

### 基本用法

```python
import torch
from visual_bge.modeling import Visualized_BGE

# 初始化模型（英文）
model = Visualized_BGE(
    model_name_bge="BAAI/bge-base-en-v1.5",
    model_weight="./models/bge-visualized-base-en-v1.5/Visualized_base_en_v1.5.pth"
)

# 初始化模型（中文/多语言）
model = Visualized_BGE(
    model_name_bge="BAAI/bge-m3",
    model_weight="./models/bge-visualized-m3/Visualized_m3.pth"
)

model.eval()

# 编码图像+文本
with torch.no_grad():
    query_emb = model.encode(
        image="./path/to/image.jpg",
        text="查询文本"
    )
    
    # 编码纯文本
    text_emb = model.encode(text="纯文本查询")
    
    # 编码纯图像
    image_emb = model.encode(image="./path/to/image.jpg")
```

### 组合图像检索示例

```python
import torch
from visual_bge.modeling import Visualized_BGE

model = Visualized_BGE(
    model_name_bge="BAAI/bge-base-en-v1.5",
    model_weight="./models/bge-visualized-base-en-v1.5/Visualized_base_en_v1.5.pth"
)
model.eval()

with torch.no_grad():
    # 查询：图像 + 文本描述
    query_emb = model.encode(
        image="./imgs/query.png",
        text="Make the background dark"
    )
    
    # 候选图像
    candi_emb_1 = model.encode(image="./imgs/candidate_1.png")
    candi_emb_2 = model.encode(image="./imgs/candidate_2.png")

# 计算相似度
sim_1 = query_emb @ candi_emb_1.T
sim_2 = query_emb @ candi_emb_2.T
print(sim_1, sim_2)
```

### 多语言多模态检索（中文）

```python
import torch
from visual_bge.modeling import Visualized_BGE

model = Visualized_BGE(
    model_name_bge="BAAI/bge-m3",
    model_weight="./models/bge-visualized-m3/Visualized_m3.pth"
)
model.eval()

with torch.no_grad():
    # 中文查询
    query_emb = model.encode(
        image="./imgs/query.png",
        text="一匹马牵着这辆车"
    )
    
    candi_emb_1 = model.encode(image="./imgs/candidate_1.png")
    candi_emb_2 = model.encode(image="./imgs/candidate_2.png")

sim_1 = query_emb @ candi_emb_1.T
sim_2 = query_emb @ candi_emb_2.T
print(sim_1, sim_2)
```

## 应用场景

1. **多模态知识检索**
   - 查询：文本
   - 候选：图像-文本对、纯文本或纯图像

2. **组合图像检索**
   - 查询：图像-文本对
   - 候选：图像

3. **多模态查询的知识检索**
   - 查询：图像-文本对
   - 候选：文本

## 常见问题

### Q1: Visual-BGE 可以用于跨模态检索（文本到图像）吗？

A1: 虽然技术上可行，但这不是推荐的使用场景。该模型专注于通过视觉能力增强混合模态检索任务。

### Q2: 安装时遇到 xformer 或 apex 错误怎么办？

A2: 不需要安装这两个包，它们对推理不是必需的。如果安装脚本要求它们，可以跳过或注释掉相关依赖。

### Q3: 模型权重文件在哪里？

A3: 下载模型后，权重文件通常命名为：
- 英文模型：`Visualized_base_en_v1.5.pth`
- 多语言模型：`Visualized_m3.pth`
